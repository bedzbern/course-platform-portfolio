## How to Use This Course

Each lesson follows the same format:

1. **Goal** — what you'll learn
2. **What You Type** — the code to type (no copy-paste!)
3. **What Each Line Means** — explanation table
4. **Try It** — a challenge to reinforce the concept
5. **What You Learned** — checklist of concepts covered

### Snapshots

Each phase has a `snapshots/phase-N-name/` folder with a fully working copy of the site at that stage. Open the `index.html` in any browser to see what your site should look like after completing that phase.

### Tips

- Type every line. Don't copy-paste. Typing builds muscle memory.
- If something breaks, check the snapshot to see what the correct code looks like.
- Do the "Try It" challenges — that's where the real learning happens.
- The final answer key is in the root `portfolio/` folder.
